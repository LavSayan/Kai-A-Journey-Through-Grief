# Step-by-Step Tutorial Guide
## Kai - A Journey Through Grief

*First-time players: Follow this guide for your first experience.*

---

## Part 1: Installation & Setup

### Step 1.1: Verify Java Installation

Before starting, ensure Java is installed on your system.

**Windows Users:**
1. Press `Windows Key + R`
2. Type: `cmd`
3. Press Enter
4. Type: `java -version`
5. You should see Java 11 or higher listed

**Mac Users:**
1. Open Terminal (Applications > Utilities)
2. Type: `java -version`
3. You should see Java 11 or higher listed

**Linux Users:**
1. Open Terminal
2. Type: `java -version`
3. You should see Java 11 or higher listed

**If Java isn't installed:**
- Visit https://www.oracle.com/java/technologies/downloads/
- Download Java 17 LTS or newer
- Follow installation instructions
- Restart your computer

### Step 1.2: Launch the Application

1. Locate the Kai application file
2. Double-click to launch
3. Wait 5-10 seconds for the menu to load
4. You'll see the main menu appear with atmospheric music

## Part 2: Main Menu Navigation

### Step 2.1: Understanding the Main Screen

When the game loads, you'll see:

```
┌─────────────────────────────────────────┐
│                                         │
│       KAI: A JOURNEY THROUGH GRIEF      │
│                                         │
│          [Start New Game]               │
│                                         │
│           [Continue]                    │
│                                         │
│             [Exit]                      │
│                                         │
└─────────────────────────────────────────┘
```

**What Each Button Does:**
- **Start New Game** → Begin a fresh story from Stage 1
- **Continue** → Resume if you've previously saved
- **Exit** → Close the application safely

### Step 2.2: Starting Your First Game

1. Move your mouse to the **"Start New Game"** button
2. Notice the button may highlight or change color (button feedback)
3. **Click** the button with your left mouse button
4. The screen will transition to the Level Selection Screen
5. Wait for the new screen to load (2-3 seconds)

## Part 3: Your First Stage

### Step 3.1: Selecting Stage 1

You'll now see the Level Screen with 5 stage options:

```
Stage 1: The Beginning
Stage 2: Growing Awareness
Stage 3: The Turning Point
Stage 4: Moving Forward
Stage 5: Resolution
```

**For First Playthrough:**
1. Click on **"Stage 1: The Beginning"**
2. The screen will transition to the first dialogue scene
3. You may see loading feedback

### Step 3.2: Your First Dialogue Scene

After a brief transition, you'll see:

**Screen Elements:**
```
┌─ Background Image ────────────────────┐
│                                       │
│  [Character Avatar or Visual]        │
│                                       │
│  "Character says: Welcome to my      │
│   journey. This is where it all      │
│   begins..."                         │
│                                       │
│  [Next →] or [Choice Buttons]        │
│                                       │
└───────────────────────────────────────┘
```

**What to Do:**
1. **Read** the dialogue text completely
2. **Observe** the visual presentation
3. **Listen** to the background music
4. **Click** the "Next" button or your choice option

### Step 3.3: Making Your First Choice

Some scenes will present dialogue choices:

**Example Screen:**
```
"How do you feel about this?"

[I'm curious and want to learn more]
        ↓
[I'm uncertain and need guidance]
        ↓
[I'm ready to accept what comes]
```

**How to Make a Choice:**
1. Read all available options
2. Think about which resonates with you
3. **Click** the option you want to select
4. The story will continue based on your choice

**Important:** There's no "wrong" choice. Each leads to meaningful dialogue.

Each stage follows this pattern:

**Scene Flow:**
```

    ↓

    ↓

    ↓
```

### Step 4.2: Key Interaction Types

**Type 1: Simple Advancement**
```
Character says something important.
[Next →] button appears.
Click to continue.
```

**Type 2: Dialogue Choices**
```
Character asks a question.
Multiple response options appear.
Click your chosen response.
Story continues with that response.
```

**Type 3: Scene Transitions**
```
Scene fades or transitions.
Information moves to next location.
New character or perspective introduced.
```

### Step 4.3: During a Stage

**What You Experience:**
- Approximately 5-10 dialogue scenes
- Mix of narrative and choices
- About 15-25 minutes of content
- Emotional narrative progression
- Atmospheric music and visuals

**What to Remember:**
- Read every line of dialogue
- Consider choices carefully
- Don't rush through scenes
- Immerse yourself in the atmosphere

## Part 7: Saving and Reloading

### Step 7.1: Automatic Saving

**When Your Game is Saved:**
- After completing each dialogue scene
- When returning to Level Screen
- Upon finishing a stage

**What Gets Saved:**
- Current stage position
- Your dialogue choices
- Character relationships/states
- Total playtime

You don't need to do anything - saving happens automatically!

### Step 7.2: Continuing Your Game

**If You Stop During Gameplay:**

1. The game saves your position
2. Close the application (or let it close)
3. Next time you launch:
   - Click "Continue" on main menu
   - You'll resume from where you stopped
   - All progress is preserved

### Step 7.3: Starting a New Game

**If You Want to Try a New Playthrough:**

1. Click "Start New Game" from main menu
2. Your previous saves are **NOT deleted**
3. You can "Continue" later to return to old saves
4. A fresh Stage 1 will begin

## Part 10: Common Questions During Gameplay

### "I clicked the button but nothing happened"

**Solution:**
1. Wait 1-2 seconds (may be processing)
2. Try clicking again
3. Check if another element needs attention first
4. Restart the stage if persistent

### "I missed something - can I go back?"

**Solution:**
- Currently, you can't go back within a stage
- Keep this in mind for replay value
- Different choices on future playthroughs show different content

### "What if I want to see other choices?"

**Solution:**
- Finish your current playthrough
- Start a New Game
- Make different choices
- Compare experiences

### "Is there a pause button?"

**Solution:**
- No pause function (visual novels typically don't have pauses)
- You can close the application anytime - it saves automatically
- Resume whenever you're ready

## Part 12: Troubleshooting Your First Session

### Issue: Screen is Black/Frozen

**Solution:**
1. Wait 5-10 seconds (loading)
2. Try moving your mouse
3. Try clicking screen center
4. Close and restart if persistent

### Issue: Text is Too Small

**Solution:**
1. Adjust your monitor resolution settings
2. Use Windows zoom (Ctrl + Plus)
3. Or adjust display scaling in Settings

### Issue: Sound Isn't Working

**Solution:**
1. Check system volume (bottom right on Windows)
2. Verify speakers/headphones are connected
3. Try adjusting Windows volume mixer
4. Restart the application

### Issue: Game Runs Slowly

**Solution:**
1. Close other applications
2. Free up RAM
3. Restart computer
4. Update graphics drivers

### Issue: I'm Stuck on a Scene

**Solution:**
1. Look for glowing or highlighted buttons
2. Try clicking different screen areas
3. Check for a "Next" button
4. Move your mouse around to find interactive elements
